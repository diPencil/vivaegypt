# 🌍 مشروع Viva Egypt Travel — الدليل الشامل والمطور
> **مستند تقني متكامل يغطي بنية المشروع، الموديولات، والتقنيات المستخدمة.**
> *آخر تحديث: 11 مايو 2026*

---

## 1. 📋 نظرة عامة (Project Overview)
مشروع **Viva Egypt Travel** هو منصة متكاملة لإدارة وحجز الخدمات السياحية (جولات، طيران، سبا، نيل كروز، وانتقالات). يتميز المشروع ببنية برمجية مرنة تعتمد على **الموديولات (Modules)** ونظام **السمات المتعددة (Multi-Themes)**.

| البند | التفاصيل |
| :--- | :--- |
| **اسم التطبيق** | Viva Egypt Travel |
| **إطار العمل الأساسي** | Laravel 10.x |
| **بيئة التشغيل** | PHP ^8.1 |
| **نمط الهندسة** | Modular Monolith (nwidart/laravel-modules) |
| **نظام الواجهات** | Blade Templates + Vite (Vue 3/JS) |
| **قواعد البيانات** | MySQL / MariaDB |

---

## 2. 🛠️ المكدس التقني (Stack)

### 2.1 Backend (PHP/Laravel)
*   **Laravel Framework:** الإصدار 10.
*   **Module Management:** `nwidart/laravel-modules` لإدارة الوحدات بشكل مستقل.
*   **Auth & Security:** Laravel Sanctum و Socialite للدخول عبر (Google/Facebook).
*   **Image Processing:** `intervention/image` لمعالجة ورفع الصور.
*   **PDF Generation:** `barryvdh/laravel-dompdf` لإصدار التقارير والفواتير.
*   **Payment Gateways:** دعم (Stripe, PayPal, Razorpay, Mollie, Fawaterk).

### 2.2 Frontend (UI/UX)
*   **Template Engine:** Laravel Blade.
*   **Asset Bundler:** Vite 4.
*   **Styling:** Bootstrap 5.2 + Vanilla CSS مع دعم كامل للـ **RTL** (العربية).
*   **Interactivity:** Vue 3 (في أجزاء محددة) و Axios لطلبات الـ AJAX.

## 3. 📂 هيكلة المجلدات الرئيسية

```text
vivaegypttravel/
├── app/                 # النواة: Controllers, Models, Middleware الأساسية.
├── Cms/                 # نظام السمات: يحتوي على 7 قوالب (theme1 -> theme7).
├── Modules/             # الموديولات الوظيفية (كل موديول له مساراته ولوحاته).
├── config/              # إعدادات المشروع (الأدوار، البوابات، إلخ).
├── database/            # التهجيرات (Migrations) والبيانات الأولية (Seeders).
├── public/              # الملفات العامة (CSS, JS, Uploads).
├── lang/                # ملفات الترجمة العامة (ar, en).
└── routes/              # المسارات العامة للويب والـ API.
```

## 5. 🚀 موديول الحجز الخاص (SpecialBooking) — *التطوير الأخير*
تم إضافة هذا الموديول لتغطية الخدمات السياحية المستقلة، ويشمل:

*   **🧖‍♂️ خدمات السبا (SPA):**
    *   إدارة خدمات السبا من لوحة التحكم (إضافة/تعديل/حذف).
    *   نموذج حجز أمامي مخصص مع كود مرجعي (Reference Code) فريد.
*   **✈️ طلبات الطيران (Flights):**
    *   استقبال طلبات تسعير الطيران وحفظها للمراجعة.
*   **🚐 الانتقالات (Transfers):**
    *   حجز خدمات التوصيل والانتقالات بين المدن والمطارات.
*   **🚢 رحلات النيل كروز (Nile Cruises):**
    *   نظام طلبات مخصص للرحلات النيلية.

## 6. 🎨 نظام السمات وتعدد الواجهات
يستخدم المشروع نظاماً فريداً في المجلد `Cms/themes/`:
*   **تعدد السمات:** يمكن للمدير تبديل شكل الموقع بالكامل بين 7 قوالب مختلفة.
*   **التخصيص:** كل سمة تملك الـ Controllers والـ Views الخاصة بها داخل مجلد الـ `Cms`.
*   **دعم RTL:** يتم تحميل ملفات CSS مخصصة (مثل `theme1-rtl.css`) تلقائياً عند اختيار اللغة العربية.

## 8. 🛠️ تعليمات التشغيل (Setup)

1.  **تثبيت الملحقات:** `composer install` و `npm install`.
2.  **إعداد البيئة:** نسخ `.env.example` إلى `.env` وضبط بيانات القاعدة.
3.  **مفتاح التطبيق:** `php artisan key:generate`.
4.  **قاعدة البيانات:** `php artisan migrate`.
5.  **روابط التخزين:** `php artisan storage:link`.
6.  **التشغيل:** `php artisan serve` و `npm run dev`.
